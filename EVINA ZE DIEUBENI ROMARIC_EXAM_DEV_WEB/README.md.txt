# Portfolio Interactif d'EVINA ZE DIEUBENI ROMARIC

## Présentation
Ce site web statique présente mon profil, mes compétences, mes projets réalisés et mes contacts. Il est conçu pour être responsive, interactif et persiste les statistiques de clics sur chaque compétence via `localStorage`.

## Architecture
- `index.html` : Structure HTML de la page
- `style.css` : Styles CSS modernes, responsives et avec effets hover et transitions
- `script.js` : Logique JavaScript pour l'interactivité, compteur de clics, affichage dynamique
- `img/` : Dossier contenant images de profil, logos, etc.

## Fonctionnalités principales
- Affichage de l'identité avec photo
- Boutons pour compétences, clics comptabilisés et description affichée dans un bloc dynamique
- Projets avec liens vers GitHub
- Informations de contact avec icônes interactives
- Design responsive avec effets CSS modernes

## Instructions
- Ouvrir le fichier `index.html` dans un navigateur moderne
- Cliquer sur les compétences pour voir leur description et compter les clics
- Les statistiques sont sauvegardées dans le navigateur avec `localStorage`
